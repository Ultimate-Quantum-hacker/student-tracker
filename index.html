Quick tour of Student Performance Tracker

Here’s where to find the most important tools:

  • Dashboard – see totals, class average, and who needs support.
  • Students / Exams / Subjects – manage names of pupils, mocks, and subjects.
  • Enter Scores – record scores for one student at a time.
  • Class Entry Mode – enter a whole class for one mock in a single grid.
  • Results & Rankings – view rank, averages, status, notes, and print reports.
  • Trends & Heatmap – see class performance over time and by subject.

Use the top-right tools to export data, manage restore points, switch themes, or start a new class based on your role.
